## Project Overview
**A comprehensive Business Intelligence solution** for a retail company transitioning to data-driven decision-making. Built with **Microsoft Fabric** and **Power BI**, this dashboard provides unified insights across sales performance, customer profitability and operational efficiency.

**Business Impact:** Enabled strategic decision-making through 14 key business questions answered across three organizational levels.
